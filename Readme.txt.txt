Getting and Cleaning Data - Project

Objective
This project sought to create a tidy dataset following the topics of this course, using the instructions provided from the assignment and checking the information included in the repository of the data source provided. 

Process and Result
The script included in the file run_analysis.R includes the following process:
1. Getting data from the online repository.
2. Reading the data and assignment to the variables to work with.
3. Merging the data and tidying the content values.
4. Selecting data according to measurements of mean and standard deviation.
5. Creating a dataset with the average of each variable for each activity and each subject.
6. Generating the result file with the data requested.
